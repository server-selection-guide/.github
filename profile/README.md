## 服務器選購前，先搞清楚一件事

選服務器的時候，大部分人的第一反應是「看配置」：CPU 幾核、內存多少、帶寬多大。

這些當然重要，但往往不是最關鍵的。

真正讓服務器用起來順不順手的，是**網絡質量**和**IP 類型**。
